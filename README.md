# DeepThought Business Analytics Assignment

This repository contains my submission for the **DeepThought Business Analytics Internship Assignment**.

## Contents

- `hyderabad_federer_25.csv`  
  Final list of 25 verified target companies with scoring, evidence, and outreach hooks.

- `Methodology.docx`  
  Research approach, filtering logic, scoring framework, and validation process used for Part A.

- `prompt.txt`  
  Prompt used during company research and scoring.

- `Part B - Sourcing Methods.docx`  
  Answer for Part B, Question 1: sourcing methods for finding Federer-profile companies across India.

- `Part B - 1000_Company_Proposal.docx`  
  Answer for Part B, Question 2: 30-day plan to build a verified 1000-company list.

## Summary

The assignment focuses on identifying specialty manufacturing companies in Hyderabad that match the required ICP, and on proposing a scalable sourcing strategy for building a larger verified company list.

My approach was:
- build a broad universe of companies,
- apply strict hard filters,
- score candidates using the provided Federer framework,
- validate borderline cases carefully,
- and document the process clearly.
